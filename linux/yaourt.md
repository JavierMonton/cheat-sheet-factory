Switches
---------
{: .-three-column}

### Installation

+ `sudo pacman -Sy yaourt` 

### Common switches

| `-S` | Install a package |
| `-Sy` | Sync database |
| `-R` | Remove a package |
| `-S [package] --noconfirm` | Install without confirmation |
