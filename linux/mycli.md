### Installation

#### Debian/Ubuntu

+ `$ sudo apt install mycli` 

#### Python

+ `$ pip install -U mycli`

#### Mac

+ `$ brew update && brew install mycli`

### Usage

+ `$ mycli -u [username] -h [host] -P [port]`
